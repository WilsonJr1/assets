# Skill: Creative Carousel from GitHub Assets

## Descrição
Esta skill cria carrosséis visuais de 5 slides para Instagram e LinkedIn a partir de um tema informado pelo usuário.

Ela utiliza os assets visuais da empresa armazenados em um repositório GitHub como referência obrigatória de identidade visual, mas não se limita a eles. A skill pode criar novos elementos gráficos (fundos, ícones, formas, composições), desde que respeitem o padrão visual da marca.

---

## Objetivo
Transformar um tema em um carrossel profissional, coerente e visualmente consistente, incluindo:

- 5 slides estruturados
- textos de cada slide
- direção visual
- legenda para LinkedIn
- legenda para Instagram
- CTA final

---

## Quando usar

Use esta skill quando quiser:
- criar carrosséis para redes sociais
- gerar criativos consistentes com a marca
- reaproveitar identidade visual existente
- escalar produção de conteúdo visual

---

## Entradas esperadas

- Tema da publicação
- Objetivo (opcional)
- Público-alvo (opcional)
- Tom (opcional)
- CTA (opcional)

---

## Saída esperada

- Título da campanha
- Estratégia do carrossel
- Slides 1 a 5:
  - objetivo
  - texto
  - direção visual
- Legenda LinkedIn
- Legenda Instagram
- CTA final

---

## Regras principais

- Sempre consultar os assets do GitHub antes de criar
- Sempre usar os assets como referência visual obrigatória
- Pode criar novos elementos visuais (fundos, ícones, shapes, etc.)
- Toda criação deve seguir o padrão da marca
- Sempre gerar 5 slides
- Sempre manter consistência visual
- Sempre adaptar para Instagram e LinkedIn

---

## Princípio central

Os assets do GitHub são a referência visual obrigatória da marca, não uma limitação criativa.

O agente deve criar novos elementos visuais sempre que necessário, desde que respeite o estilo, cores e linguagem dos assets existentes.
